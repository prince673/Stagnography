# Stagnography
Stagnography_using_python
A Collection of Steganography Techniques for Secure Data Hiding.

Overview
This project is a collection of tools and techniques for implementing steganography, the practice of hiding secret data within other non-secret data such as images, audio, and video files. It includes methods for encoding and decoding hidden messages, aiming to provide both beginners and advanced users with practical examples of secure data hiding.

Features
Image-based Steganography:

Hides messages in image files using Least Significant Bit (LSB) technique.
Support for multiple image formats (PNG, BMP, etc.).
Audio-based Steganography:

Embeds secret data within audio files without affecting the perceptual quality.
Video-based Steganography:

Encrypts messages within video frames for added security.
Encryption Support:

Optionally encrypt hidden messages using AES for an extra layer of protection.
Technologies
Languages: Python, C++
Libraries: Wave 
File Formats Supported: PNG, BMP, WAV, MP4
